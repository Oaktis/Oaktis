## Running on Kaggle

1. Go to [Kaggle Notebooks](https://www.kaggle.com/code)
2. Click "New Notebook"
3. Upload the notebook file
4. Run the cells

## Running Locally

1. Install Jupyter:

```bash
pip install jupyter
```

2. Start Jupyter:

```bash
jupyter notebook
```

3. Open a notebook and run the cells
